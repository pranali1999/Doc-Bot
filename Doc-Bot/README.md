# flask-chatbot
python 3.6 

Dependencies :  

             Flask
             
             chatterbot
             
             SQLAlchemy
             
             Speech Recognization(for speech to text)   
             
             pyttsx3(text to speech)
               
Local Setup:
 1. Open command prompt and locate folder. run 'pip install -r requirements.txt'
 2. Run train.py
 3. Run run.py
 4. Demo will be live at http://localhost:5000/
