# Doc-Bot
  
flask-chatbot
python 3.6

Dependencies :

         Flask
         
         chatterbot
         
         SQLAlchemy
         
         Speech Recognization(for speech to text)   
         
         pyttsx3(text to speech)
Local Setup:

Open command prompt and locate folder. run 'pip install -r requirements.txt'
Run train.py
Run run.py
Demo will be live at http://localhost:5000/
